# Logistics Company - Vehicle Management System

OOP assignment demonstrating inheritance in C# .NET.

## Structure
- **Vehicle** (base class)
  - **Car** → **SUV** and **Cabriolet** (multi-level inheritance)
  - **Motorcycle**
  - **Truck**
  - **Bicycle** (no engine)

## Features
- Electric SUV implementation
- All vehicles have unique StartEngine() sounds
- Demonstrates inheritance, polymorphism, and method overriding
